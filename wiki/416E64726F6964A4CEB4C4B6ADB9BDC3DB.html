Androidの環境構築

SDK のダウンロード

Android Studio をインストールする．

  • Do not import setting をチェック
  • Android Studio Setup Wizard が始まるので Next
  • Install Type は Standard を選択して Next
  • Select UI Theme は IntelliJ を選択して Next

Android Studio を起動して Configure -> SDK Manager で SDK をダウンロードする．
Android API と必要なバージョンにチェックをいれてダウンロードする．
SDK Tools タブを開き，右下の Show Package Details をチェックしてリスト内の Android SDK Build-Tools して必要なバージョンをチェックしてダウンロードする．

JDK のインストール

Java SE Development Kit をダウンロードする．

  • Accept License Agreement をチェックして対応する OS を選択，ダウンロードしてインストール

NDK のダウンロード

Android NDK をダウンロードして解凍，適当な場所に配置する．(https://developer.android.com/ndk/downloads?hl=ja)

ANT のダウンロード

Apache から Ant をダウンロードする．(https://ant.apache.org/bindownload.cgi)
中段から必要なバージョンを選択してダウンロード，解凍して適当な場所を配置する．
