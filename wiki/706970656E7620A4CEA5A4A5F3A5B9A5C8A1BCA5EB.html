pipenv のインストール

pipenv をインストールする

$ pip install pipenv

pip を使用するとエラーが出る

pipenv をインストールすると pip を使用するときにエラーが出るようになってしまう場合がある．

$ pip --version
Traceback (most recent call last):
  File "/usr/bin/pip", line 9, in <module>
   from pip import main
ImportError: cannot import name main

原因として pip のバージョン 10 からディレクトリの構造が変わったようである．(http://icchy.hatenablog.jp/entry/2018/04/17/064443)
Ubuntu 18.04 LTS ではデフォルトで pip のバージョンは 9.0.1 であった．

そのため一度 pip をアンインストールする．

$ sudo apt --purge remove python-pip
$ python -m pip uninstall pip

そして apt を使用せずに pip をインストールする．

$ curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
$ python get-pip.py --user

エラーが表示された場合は PATH を設定する．

WARNING: The scripts pip, pip2 and pip2.7 are installed in '/home/viviria/.local/bin' which is not on PATH.
Consider adding this directory to PATH or, if you prefer to suppress this warning, use --no-warn-script-location.

$ vi ~/.profile
PATH="$PATH:/home/[USER_NAME]/.local/bin"
$ source ~/.profile

これで pip を使用できる．

$ pip --version
pip 19.3.1 from /home/[USER_NAME]/.local/lib/python2.7/site-packages/pip (python 2.7)

この方法で pip をインストールした場合は pipenv はすでにインストールされているようである．

$ pipenv --version
pipenv, version 2018.11.26
